# ParkOps 智慧停车运营平台

面向停车场运营的智慧平台，聚焦“停车资源运营 + 收费结算 + 设备运维 + 数据分析”。该仓库提供一套可以写进简历的 Java 实习项目骨架，并规划 1 个月完成路线图。

## 目标与定位

- **核心定位**：服务停车场运营者（B 端），统一管理车场、车位、计费、订单、优惠、设备与报表。
- **技术定位**：Java + Spring Boot + REST API，后续可扩展到微服务。
- **可交付物**：基础平台 API、最小可用演示、可在简历中说明的功能与架构亮点。

## 1 个月交付计划（周拆解）

### 第 1 周：基础架构 + 核心资源建模
- 统一项目结构：Spring Boot、统一异常、统一返回结构
- 建模：停车场、车位、车辆、订单、收费规则
- 基础 API：停车场/车位 CRUD、车位状态管理
- 健康检查与运行说明

### 第 2 周：计费与订单闭环
- 计费策略（时段/阶梯/封顶）
- 入场/出场流程（进出场记录 + 订单生成）
- 订单支付状态流转（待支付/已支付/异常）
- 基础统计：日收入、车位利用率

### 第 3 周：运营功能完善
- 优惠券与折扣规则
- 设备/闸机管理接口（模拟数据接入）
- 报表接口：按车场/时段维度统计
- 数据权限与角色（运营管理员、收费员、巡检员）

### 第 4 周：演示 + 简历素材
- 对接前端或 API 文档演示
- 统一接口文档（OpenAPI/Swagger）
- 性能与可靠性说明（缓存、限流、幂等）
- 简历描述（功能清单 + 技术栈 + 亮点）

## 当前已提供的基础能力

- Spring Boot 启动骨架
- `GET /api/health` 健康检查接口
- 基础配置（端口、应用名）
